# AI楽曲批評ツール

生成AIで制作した楽曲を、**音楽プロデューサー・音楽評論家・一般リスナーの3つの視点**から分析・批評・評価するWebツールです。音楽ファイルと歌詞を入れるだけで、曲調・構成・ミックス・歌詞を解析し、小説のように読める批評文と具体的な改善案を生成します。

## 主な機能

- **音響解析(すべて端末内で実行)**
  - テンポ(BPM)推定・リズムの安定度
  - キー(調)判定(24調 + 長調/短調)
  - ダイナミックレンジ・音圧・クリッピング(音割れ)検出
  - 7帯域の周波数バランス/音色の明るさ
  - ステレオの広がり
  - 曲の構成(セクション数・イントロ長・サビの対比・終わり方)
- **歌詞解析**:構成(サビの有無)・語彙の豊かさ・感情語/情景語・常套句検出・韻の傾向
- **批評の口調**:甘口/一般/辛口 の3段階
- **ジャンル別評価基準**:ポップス/ロック/バラード/EDM/ヒップホップ/ジャズ/アニソン/劇伴 ほか
- **出力**:総合ランク(S〜D)・5項目レーダーチャート・読み物風の批評文・改善案・技術データ
- **便利機能**:縦書き表示/コピー/テキスト保存/印刷/批評履歴(端末内保存)
- **PWA対応**:ホーム画面に追加すればアプリとして起動、オフラインでも動作

## セキュリティ・プライバシー

- 音源・歌詞は**ブラウザ内(端末内)だけで解析**され、外部サーバーへは一切送信されません
- Content-Security-Policy によりすべての外部リソース読み込みを遮断(完全自己完結)
- 画面に表示するテキストはすべて `textContent` 経由で挿入(HTMLインジェクション対策)
- 外部ライブラリ・CDN・アナリティクス不使用

## ローカルでの確認方法

「ローカルで確認.command」をダブルクリックするか、ターミナルで:

```bash
cd 作品批評ツール
python3 -m http.server 8934
# → http://localhost:8934 を開く
```

※ `index.html` を直接開いても動作しますが、PWA(オフライン対応)はHTTPS または localhost でのみ有効です。

## 公開方法(無料)

静的サイトなので、フォルダごとアップロードするだけで公開できます。

### GitHub Pages
1. GitHubで新しいリポジトリを作成
2. このフォルダの中身をすべてアップロード(`index.html` がルートに来るように)
3. リポジトリの Settings → Pages → Branch を `main` にして保存
4. 数分後、`https://ユーザー名.github.io/リポジトリ名/` で公開されます

### Netlify(もっと簡単)
1. https://app.netlify.com/drop を開く
2. このフォルダをドラッグ&ドロップ → 即公開

公開後、スマホで開いて「ホーム画面に追加」すればアプリとして使えます。

## ファイル構成

```
index.html            画面本体
css/style.css         デザイン
js/analyzer.js        音響解析エンジン(FFT/BPM/キー/構成)
js/lyrics.js          歌詞解析エンジン
js/database.js        批評データベース(基準・表現・改善案)
js/critique.js        批評文生成エンジン
js/app.js             UI制御・履歴・保存
manifest.webmanifest  PWA設定
sw.js                 Service Worker(オフライン対応)
assets/               アイコン・ファビコン
```
